<h2><a href="https://leetcode.com/problems/valid-palindrome-ii/">680. Valid Palindrome II</a></h2>
<h3>Difficulty: Easy</h3>
<hr>

Given a string `s`, return `true` *if the* `s` *can be palindrome after deleting **at most one** character from it*.

**Example 1:**

```
Input: s = "aba"
Output: true
```

**Example 2:**

```
Input: s = "abca"
Output: true
Explanation: You could delete the character 'c'.
```

**Example 3:**

```
Input: s = "abc"
Output: false
```

**Constraints:**

* `1 <= s.length <= 105`
* `s` consists of lowercase English letters.
